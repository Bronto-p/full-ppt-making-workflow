# PPT Workflow Skills

Standalone Codex skills for a simplified PPT workflow.

## Skills

- `ppt-plan-order`: create a consistent `ppt_plan.md` from client materials.
- `ppt-style-sample`: create a three-slide sample round from `ppt_plan.md`, using codex-ppt-style slide job, subagent, and image generation machinery.
- `ppt-full-production`: produce the final image-based PPTX from `ppt_plan.md` and approved sample references.

These are skills, not a plugin. Install by copying each folder under `skills/` into `~/.codex/skills/`.
